## 更新日志说明

### 类型说明
- **新增**: 新功能
- **修改**: 现有功能的更改
- **已弃用**: 即将删除的功能
- **移除**: 已删除的功能
- **修复**: 问题修复
- **安全**: 安全相关的更改

### 版本说明
- 遵循语义化版本 (SemVer)
- 格式: MAJOR.MINOR.PATCH
- MAJOR: 不兼容的 API 更改
- MINOR: 向后兼容的功能新增
- PATCH: 向后兼容的问题修复
